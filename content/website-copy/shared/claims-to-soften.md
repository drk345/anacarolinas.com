# Afirmaciones a suavizar o evitar

## Términos prohibidos

| Evitar | Alternativa aprobada |
|--------|---------------------|
| "Entrenadora de Cerebros" | "Coach integrativa · neurociencia del bienestar · práctica somática" |
| "cambio neurológico real" | "entrenar nuevas respuestas de forma consciente y sostenida" |
| "entrenamiento neurológico personalizado" | "entrenamiento personalizado basado en neuroeducación, práctica somática y coaching" |
| "arreglarte" / "no estás rota" | "Aquí partimos de tus recursos, tu experiencia y tu capacidad de transformación." |
| "cura" / "curar" | (eliminar; usar "acompañar", "entrenar", "transformar") |
| "tratamiento clínico" | (solo en límite terapéutico, con contexto correcto) |
| "resultado garantizado" | (eliminar; nunca garantizar resultados) |
| "diagnóstico" | (solo en contexto clínico correcto — no usar para describir sesiones) |
| "no estás rota" / "no estas rota" | (eliminar en todas las variantes) |
| urgencia falsa: "¡Solo X plazas!" | (no usar sin información real verificada) |

## Afirmaciones de experiencia a manejar con cuidado

- "Más de 15 años de experiencia" — verificada; usar con esa formulación exacta
- Estadísticas de resultados (clientes, tasas de cambio, etc.) — NO inventar ninguna
- Credenciales específicas (máster, certificación, etc.) — NO inventar; confirmar con Ana
- "programa basado en neurociencia" — OK si se aclara que es neuroeducación/aplicada, no investigación clínica original

## Promesas que requieren matiz

- "transformación profunda" — OK con moderación; evitar "garantizada"
- "cambio real y sostenido" — OK; evitar "permanente"
- "aprender a regular el sistema nervioso" — OK como formulación educativa
- "reprogramar el cerebro" — EVITAR; demasiado neurocientíficamente inexacto
